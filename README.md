# 基于区块链的隐私保护基因分析系统

## 系统功能
- 检索可供检测的疾病
- 用户上传比较基因
- 医疗检测机构上传致病基因
- 计算节点提供算力，计算基因交集
- 用户查询结果
- 获取最新的区块链
